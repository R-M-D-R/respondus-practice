# 🛡️ Proctoring Certification Training Site

This project is a **portable, reusable training system** designed to prepare students for proctored online assessments using tools like:

- Respondus LockDown Browser
- Respondus Monitor
- Webcam + microphone proctoring systems
- Secondary camera (cell phone recording)

---

# 🎯 Purpose

This training system exists to:

✅ Reduce student confusion  
✅ Prevent failed proctoring attempts  
✅ Standardize expectations across courses/schools  
✅ Teach correct testing setup BEFORE assessments  

---

# 🧠 Design Philosophy

This site is built around one core principle:

> **Reduce ambiguity as much as possible.**

Students often fail proctoring not because of cheating, but because of:

- unclear expectations  
- poor setup  
- misunderstanding requirements  

This system fixes that through:

- visual examples
- explicit rules
- repeated structure
- enforced interaction

---

# 🔁 How It Works

## ✅ Student Flow

1. Student clicks training link from LMS (Canvas, etc.)
2. Student completes training (this site)
3. Student confirms setup via checklist
4. Student returns to LMS
5. Student completes certification attempt

---

## 🔗 Dynamic Return System

This site uses a **URL parameter** to send students back to the correct course location.

### Example: https://your-site.com/index.html?return=https://canvas.com/your-assignment

The "Return to Canvas" button dynamically redirects students to the correct page.

✅ One HTML file works for ALL courses  
✅ No need to edit the file per class

# 🧩 Features

## ✅ Visual Training System

- Camera setup diagrams  
- Lighting examples (good vs bad)  
- Workspace examples  
- Behavior warnings  

---

## ✅ Interactive Checklist

Students must:

- Check all setup requirements  
- Confirm understanding  
- Unlock the return button  

This prevents passive scrolling and forces engagement.

---

## ✅ Behavior Enforcement

The button remains locked until:

- All checklist boxes are checked  
- Student accepts the commitment statement  

---

## ✅ Reset on Reload

Checklist automatically resets on page load to prevent students from skipping steps.
